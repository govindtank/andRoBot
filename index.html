Androbot

Scorpius is a HTTP based vision and voice controlled robotic software for Android. It features voice recognition, voice synthesis, remote surveillance, face detection, image & sensor based motion detection and Google Maps based locating. When connected to the Arduino ADK based robot, it can also be operated in line following, object tracking and remote controlled mode. It can take photos in normal, panoramic and time-lapse modes.
